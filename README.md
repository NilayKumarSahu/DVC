# Command Prompt

```
mkdir DVC
```
# change directory

```
cd DVC
```
# Open Vs code with same directory

```
code .
```

# Create Environment

```
conda create --prefix venv2 python = 3.8
```
# Create gitignore file

```
touch .gitignore
```
# Create readme file

```
touch README.md
```

# Activate environment

```
conda activate ./venv2
```

# Install requirements.txt

```
pip install -r requirements.txt
```

# Initialize dvc

```
dvc init
```
